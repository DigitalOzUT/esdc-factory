# Image tools

This directory holds scripts, which are used for image preparation and post-install procedures.

## prepare-image

Run the following commands:
	
	curl -o /usr/sbin/prepare-image <URL>
	chmod +x /usr/sbin/prepare-image
	./prepare-image

### Supported platforms

* CentOS 6
* CentOS 7
